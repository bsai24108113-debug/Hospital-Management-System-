Project Description
A Java Swing–based Hospital Management System that manages patients, doctors, departments, and staff efficiently. The application provides a user‑friendly GUI with CRUD operations, secure login, database connectivity, and real‑time record viewing to streamline hospital workflows and reduce manual paperwork.

Hospital Management System
Overview
This project is a desktop-based Hospital Management System developed in Java (Swing). It is designed to help hospitals manage patient records, employee information, departments, and administrative tasks through a simple and interactive graphical user interface.

Features
User authentication (Login system)
Add, update, delete, and view patient records
Manage doctors, staff, and departments
Database integration using MySQL
Clean and beginner-friendly Swing GUI
Modular and easy-to-extend code structure
Technologies Used
Java (JDK 8+)
Java Swing (GUI)
MySQL Database
JDBC
NetBeans / IntelliJ / Eclipse (any IDE)
Project Structure
Login – Handles user authentication
Patient – Patient record management
Department – Department information
Employee – Staff and doctor records
Database – JDBC connection handling
How to Run
Clone the repository
Open the project in any Java IDE
Configure MySQL database and update JDBC credentials
Import required libraries (JDBC, rs2xml if used)
Run the main class
Database Setup
CREATE DATABASE hospital_management_system;
USE hospital_management_system;
-- Create required tables (patients, doctors, login, etc.)
